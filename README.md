# To-Do List

A simple CRUD to-do list app built with vanilla HTML, CSS and JavaScript.

## Features

- **Add** tasks by typing and pressing Add or Enter
- **Complete** tasks by clicking on them (strikethrough)
- **Delete** tasks with the Slett button
- **Persistent** — tasks are saved in localStorage and survive a page refresh

## What I learned

- CRUD operations (Create, Read, Update, Delete)
- DOM manipulation with vanilla JavaScript
- Saving and loading data with localStorage
- Event listeners and dynamic element creation

## How to run

1. Clone the repo
```bash
git clone https://github.com/669830/todo-list.git
```
2. Open `index.html` in your browser

No installs, no dependencies — just open and use.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
